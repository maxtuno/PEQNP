Diophantine Equations

Let be solve 

from peqnp import *

engine(10)

x = integer()
y = integer()

assert x ** 3 - x + 1 == y ** 2

assert x != 0
assert y != 0

while satisfy(normalize=True):
    print('{0} ** 3 - {0} + 1, {1} ** 2'.format(x, y))

5 ** 3 - 5 + 1, 11 ** 2
1 ** 3 - 1 + 1, 1 ** 2
3 ** 3 - 3 + 1, 5 ** 2
